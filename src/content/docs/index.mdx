A curated collection of animated React components designed for modern web applications. Built with TypeScript, Framer Motion, and Tailwind CSS, these components are crafted for seamless integration, smooth animations, and exceptional user experience.

## Available Components

- **[Marquee](/docs/components/marquee)** - Smooth scrolling text and content marquee
- **[Animated Number](/docs/components/number/animated-number)** - Smooth number transitions and counters
- **[Transition Panel](/docs/components/tabs/transition-panel)** - Animated content panels with smooth transitions
- **[Scroll Active Section](/docs/components/scroll/scroll-active-section)** - Scroll-triggered content sections
- **[Velocity Text](/docs/components/text/velocity-text)** - Dynamic text animations based on scroll velocity
- More components coming soon...


### UI Blocks
- **[Testimonial](/docs/blocks/testimonial/x)** - Social media-style testimonial blocks
- More blocks coming soon...

## Getting Started

1. **Browse Components** - Explore our component library to find what you need
2. **Copy Code** - Use the copy button to grab the component code
3. **Install Dependencies** - Add required packages like `framer-motion` if needed
4. **Customize** - Modify styles and behavior to match your design system
