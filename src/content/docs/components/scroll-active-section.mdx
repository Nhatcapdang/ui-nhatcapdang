# Scroll Active Section

A powerful scroll-based component that displays content on the left side while showing an active item preview on the right. Perfect for showcasing features, timelines, or step-by-step processes.

## Features

- **Scroll-based activation**: Items become active as they scroll into view
- **Smooth animations**: Uses Framer Motion for fluid transitions
- **Progress indicator**: Visual progress bar and dots
- **Responsive design**: Adapts to mobile and desktop layouts
- **Customizable**: Extensive styling options and content flexibility
- **TypeScript support**: Fully typed with comprehensive interfaces

## Usage

```tsx
import { ScrollActiveSection, ScrollItem } from '@/components/scroll-active-section'

const items: ScrollItem[] = [
  {
    id: 'item-1',
    title: 'First Item',
    subtitle: 'Optional subtitle',
    description: 'Detailed description of the first item...',
    image: 'https://example.com/image1.jpg',
    content: <div>Optional custom content</div>
  },
  {
    id: 'item-2',
    title: 'Second Item',
    description: 'Description of the second item...',
  }
]

export function MyComponent() {
  return (
    <ScrollActiveSection 
      items={items}
      showProgress={true}
    />
  )
}
```

## API Reference

### ScrollActiveSection Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `items` | `ScrollItem[]` | - | Array of items to display |
| `className` | `string` | - | Additional CSS classes for the container |
| `leftClassName` | `string` | - | CSS classes for the left content area |
| `rightClassName` | `string` | - | CSS classes for the right preview area |
| `itemClassName` | `string` | - | CSS classes for individual items |
| `activeItemClassName` | `string` | - | CSS classes for the active item preview |
| `showProgress` | `boolean` | `true` | Whether to show progress indicators |

### ScrollItem Interface

```tsx
interface ScrollItem {
  id: string
  title: string
  subtitle?: string
  description: string
  image?: string
  content?: React.ReactNode
}
```

## Examples

### Basic Usage

```tsx
const basicItems: ScrollItem[] = [
  {
    id: 'step-1',
    title: 'Getting Started',
    description: 'Learn the basics of our platform...'
  },
  {
    id: 'step-2',
    title: 'Advanced Features',
    description: 'Explore powerful capabilities...'
  }
]

<ScrollActiveSection items={basicItems} />
```

### With Images and Custom Content

```tsx
const richItems: ScrollItem[] = [
  {
    id: 'feature-1',
    title: 'AI-Powered Analysis',
    subtitle: 'Next-generation processing',
    description: 'Advanced algorithms provide intelligent insights...',
    image: '/images/ai-analysis.jpg',
    content: (
      <div className="flex gap-2">
        <Badge>Machine Learning</Badge>
        <Badge>Real-time</Badge>
      </div>
    )
  }
]

<ScrollActiveSection 
  items={richItems}
  className="bg-gradient-to-b from-background to-muted/20"
  showProgress={true}
/>
```

### Custom Styling

```tsx
<ScrollActiveSection 
  items={items}
  className="min-h-screen"
  leftClassName="space-y-32"
  rightClassName="lg:top-32"
  itemClassName="p-8 rounded-lg border"
  activeItemClassName="shadow-2xl border-primary"
  showProgress={false}
/>
```

## Customization

The component is highly customizable through CSS classes and supports:

- **Progress indicators**: Toggle with `showProgress` prop
- **Custom spacing**: Adjust gaps between items
- **Responsive behavior**: Different layouts for mobile/desktop
- **Animation timing**: Modify transition durations
- **Color schemes**: Adapt to your design system

## Accessibility

- **Keyboard navigation**: Fully accessible via keyboard
- **Screen readers**: Proper ARIA labels and semantic HTML
- **Focus management**: Clear focus indicators
- **Reduced motion**: Respects user motion preferences

## Browser Support

- Chrome 90+
- Firefox 88+
- Safari 14+
- Edge 90+

Requires support for Intersection Observer API and CSS Grid.
