# Components

Welcome to the Components section! Here you'll find reusable UI components that you can integrate into your projects.

## Available Components

- [Scroll Active Section](/docs/components/scroll-active-section) - Scroll-based component with content on the left and active item display on the right
- [Marquee](/docs/components/marquee) - Smooth scrolling marquee component
- [Animated Number](/docs/components/number/animated-number) - Smooth animated number transitions
- [Transition Panel](/docs/components/tabs/transition-panel) - Smooth panel transitions
- [Velocity Text](/docs/components/text/velocity-text) - Scroll-based velocity text animation
