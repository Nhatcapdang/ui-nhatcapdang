## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `number` | - | The numeric value to animate to |
| `className` | `string` | - | Additional CSS classes to apply |
| `springOptions` | `SpringOptions` | - | Framer Motion spring animation options |
